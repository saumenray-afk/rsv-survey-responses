RSV CLUB HOUSE SURVEY - QUICK START GUIDE
═══════════════════════════════════════════════════════════════

🚀 GET YOUR SURVEY APP RUNNING IN 10 MINUTES

═══════════════════════════════════════════════════════════════

STEP 1: CREATE GITHUB ACCOUNT (2 minutes)
────────────────────────────────────────────
1. Go to https://github.com
2. Click "Sign up"
3. Create free account
4. Verify your email

═══════════════════════════════════════════════════════════════

STEP 2: CREATE RESPONSE STORAGE REPOSITORY (1 minute)
────────────────────────────────────────────
1. Click "+" (top right) → "New repository"
2. Repository name: rsv-survey-responses
3. Description: RSV Club House Survey Responses
4. Select: ⚫ Private (IMPORTANT - protects resident data)
5. ✅ Check "Add a README file"
6. Click "Create repository"

═══════════════════════════════════════════════════════════════

STEP 3: CREATE RESPONSES FOLDER (30 seconds)
────────────────────────────────────────────
1. In your new repository, click "Add file" → "Create new file"
2. In the file name box, type: responses/.gitkeep
3. Scroll down, click "Commit new file"

═══════════════════════════════════════════════════════════════

STEP 4: GENERATE ACCESS TOKEN (2 minutes)
────────────────────────────────────────────
1. Click your profile picture (top right) → Settings
2. Scroll to bottom → Click "Developer settings"
3. Click "Personal access tokens" → "Tokens (classic)"
4. Click "Generate new token" → "Generate new token (classic)"
5. Note: "RSV Survey App"
6. Expiration: 90 days (or as needed)
7. Select scopes:
   ✅ repo (Full control of private repositories)
8. Scroll down, click "Generate token"
9. ⚠️ COPY THE TOKEN NOW (starts with ghp_...)
   You won't see it again!

═══════════════════════════════════════════════════════════════

STEP 5: CONFIGURE APP FILES (2 minutes)
────────────────────────────────────────────

Open app.js in a text editor and find this section:

const GITHUB_CONFIG = {
    owner: 'YOUR_GITHUB_USERNAME',
    repo: 'rsv-survey-responses',
    token: 'YOUR_GITHUB_TOKEN',
    branch: 'main'
};

Replace:
- YOUR_GITHUB_USERNAME → your actual GitHub username
- YOUR_GITHUB_TOKEN → the token you copied in Step 4

Example:
const GITHUB_CONFIG = {
    owner: 'rajesh-kumar',
    repo: 'rsv-survey-responses',
    token: 'ghp_AbCdEf1234567890XyZ...',
    branch: 'main'
};

Save the file!

═══════════════════════════════════════════════════════════════

STEP 6: HOST THE APP (3 minutes)
────────────────────────────────────────────

OPTION A - NETLIFY (EASIEST, RECOMMENDED)
─────────────────────────────────────────
1. Go to https://www.netlify.com
2. Click "Sign up" → Sign up with GitHub
3. After login, click "Add new site" → "Deploy manually"
4. Drag and drop ALL your survey files (index.html, app.js, etc.)
5. Wait 30 seconds for deployment
6. Your app is live! Copy the URL (e.g., https://rsv-survey-xyz.netlify.app)

OPTION B - GITHUB PAGES
─────────────────────────────────────────
1. Create NEW repository: rsv-survey-app
2. Make it PUBLIC
3. Upload all files
4. Go to Settings → Pages
5. Source: Deploy from branch → main → / (root)
6. Save, wait 2 minutes
7. URL: https://YOUR_USERNAME.github.io/rsv-survey-app/

OPTION C - TEST LOCALLY
─────────────────────────────────────────
1. Download Python: https://www.python.org/downloads/
2. Open Command Prompt/Terminal in survey folder
3. Run: python -m http.server 8000
4. Open browser: http://localhost:8000
5. (Only accessible on your computer)

═══════════════════════════════════════════════════════════════

STEP 7: TEST YOUR APP (2 minutes)
────────────────────────────────────────────
1. Open your app URL on your phone
2. Click "Install" if prompted
3. Fill test data:
   - Villa: TEST-101
   - Name: Test User
   - Phone: 9999999999
4. Complete a few choice sets
5. Click Submit
6. Go to GitHub → rsv-survey-responses → responses folder
7. You should see: response_TEST101_xxxxx.json
8. Click the file to view the data

✅ If you see the file, YOUR APP IS WORKING!

TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════

❌ "Responses not saving"
   → Check GitHub token hasn't expired
   → Verify token in app.js is correct
   → Responses still save locally as backup

❌ "Can't install app"
   → Use Chrome browser
   → Make sure URL is HTTPS
   → Clear cache and try again

❌ "Error when submitting"
   → Check internet connection
   → Responses save locally
   → Re-submit will be attempted
